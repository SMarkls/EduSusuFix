# EduSusuFix
Тёмная тема и удаление иконок курсов на сайте edu.susu.ru
##
* Любите заливать лабы по ночам?
* Слепните при входе на сайт после захода _солнца_?
* Не нравятся непонятные иконки возле названия курсов?
Тогда `ЕдуСусуФикс` для **Вас**!  
Расширение добавляет темную тему на сайт, а также удаляет иконки курсов.

# Установка
**Расширение работает в браузерах на движке Chromium, то есть на Yandex.Browser, Chrome, Opera, как себя будет вести расширение в Edge и FireFox - без понятия!**
Пока найду желание разбираться, как заливать расширения в магазины расширений установка происходит следующим образом:
* Нажимаете на зеленую кнопку `Code` в правом верхнем углу репозитория
* Нажимаете `Download ZIP`, скачанный архив распаковываете куда хотите
* Далее заходим в управление расишрениями `opera://extensions` или `chrome://extensions`
* Включаем _Режим разработчика_ и в появившейся панели выбираем `загрузить распакованное расширение`
* Указываем путь до той папки, где лежит `manifest.json`.
* **Готово! Наши глазки больше болеть не будут =)**
